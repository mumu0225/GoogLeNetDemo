# GoogLeNetDemo  
File "split.py" changes the file structure of the image in the 101_ObjectCategories to the same as that of ImageNet, and contains a train file and val file.  
The folder format should be as follows  
The dataset name/  
  ├──train/  
  │  ├──class1/  
  │  │  ├──image1.jpg  
  │  │  ├──image2.jpg  
  │  │  └── ...  
  │  ├──class2/  
  │  │  ├──image1.jpg  
  │  │  ├──image2.jpg  
  │  │  └── ...  
  │  └── ...  
  └──val/  
     ├──class1/  
     │  ├──image1.jpg  
     │  ├──image2.jpg  
     │  └── ...  
     ├──class2/  
     │  ├──image1.jpg  
     │  ├──image2.jpg  
     │  └── ...  
     └── ...
